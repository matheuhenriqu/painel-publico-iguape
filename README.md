# Painel Publico de Contratos - Iguape/SP

Versao estatica do painel publico para publicacao no GitHub Pages.

URL esperada apos publicar:

`https://matheuhenriqu.github.io/painel-publico-iguape/`

## Automacao

- Atualizacao automatica da base publica a cada hora via GitHub Actions
- Validacao automatica do JSON publico, da interface e das protecoes do frontend
- Publicacao automatica no GitHub Pages apos push na branch `main`
- Execucao manual disponivel em `Actions` para disparar atualizacao ou validacao sob demanda
